#' タイトル．`help(package = mypackage)` で表示される説明文
#'
#' 概要．簡潔に．
#'
#' 詳細．任意．しばしば長文．
#'
#' @param 引数 引数の説明
#' @return 返り値の説明
#' @examples
#' 関数の使い方を実行可能なRコードで記述．
#' 適宜改行．
#' R CMD CHECK で確認される．
#' @aliases 関数名を変更した時に古い関数名を記載
#' @name
#' @export
#' @rdname
#' @description
#' @details
  • @param
    • 引数の目的，型などについて説明．
    • 大文字で始めピリオドで終わる．
    • 複数の引数に同じ説明をつけたい時はカンマで区切る (例: #' @param x,y numeric vectors)． カンマ前後にスペースは使わない．

markdownの文法を使いたい場合は DESCRIPTION に

Roxygen: list(markdown = TRUE)

を追記

f<-data_frame(id=c(1:4), name=c("ringo", "itigo", "banana", "kiui"), rank=c(1,2,2,1))
p<-data_frame(rank=c(1,2,3), price=c(220,440,880))
